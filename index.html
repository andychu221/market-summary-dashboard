dashboard Summary Tab

The Summary tab is the landing page of the dashboard. It provides a snapshot of market performance across different sectors (Benchmark, Watchlist, Peers).

Key Features:

  • As of Date: Change this date to view historical snapshots of the table. Note: Charts will update to reflect the selected end date.
  • Search: Use the text box to filter the table rows by Ticker symbol or Company Name.
  • Customize Columns: Click to toggle visibility or reorder columns via drag-and-drop. You can also rename column headers.
  • Manage Tickers: Add or remove tickers from the three main sections. You can also rename the section titles.
  • Interactive Headers: Click on column headers to sort. Click on the Date header to change the date display format. Click on Since... header to change the reference start date for that column.
  • Row Click: Clicking on any row will open a detailed chart for that specific ticker.

show_chart Chart Tab

This tab allows for detailed comparative analysis of stock performance over time.

Navigation:

  • Sub-tabs: Use the buttons at the top (e.g., 1-Week View, Main, Peers) to switch between different preset chart views. Double-click a tab to rename it.

Controls:

  • Tickers Input: Edit the comma-separated list of tickers to change which stocks are displayed on the chart.
  • Time Controls: Select preset ranges (1w, 1m, 3m, YTD, etc.) or choose 'Custom' to define specific start and end dates.
  • Update Button: Must be clicked after changing tickers or dates to refresh the chart.

Chart Types:

  • Indexed Price: Most tabs show rebased performance (starting at 0%) for easy comparison.
  • Trading Volume (Chart-7): A specialized grid view showing Price vs Volume for up to 4 tickers.

article News Tab

Access the latest news and AI-generated summaries for watchlist companies.

How to use:

  • Select Company: Use the dropdown to choose a specific company. Selecting 'All' aggregates news from all sources.
  • Filter Source: Narrow down news by provider (e.g., Reuters, Bloomberg).
  • Date Range: Click to select a start and end date to filter news items.
  • AI Summary: If available, the first item in the list will be an AI-generated summary of recent events.
  • Reading: Click a news item on the left to read the full content on the right pane.

table_view Data Tab

View and export the raw data underlying the dashboard.

  • Filtering: Set Start and End dates to limit the data range.
  • Pivot View: Data is pivoted with Dates as rows and Tickers as columns for easy analysis.
  • Export: Use the "Export to Excel" button to download the full dataset including Price and Volume sheets.

email Reporting

Generate a weekly report suitable for email distribution.

  • Click Generate Report in the top header.
  • The system captures current chart states and compiles AI summaries.
  • Editable Content: You can click inside the generated preview to edit text, delete sections, or add comments before downloading.
  • Download Draft: You can download the report as an Outlook .eml file to send later.
  • Edit Fields: You can pre-fill To, Cc, and Subject lines before downloading.

dashboard 總覽 (Summary Tab)

此頁面是儀表板的首頁，提供不同板塊（基準、觀察名單、同業）的市場表現快照。

主要功能：

  • 日期選擇 (As of): 更改此日期可查看歷史數據快照。注意：圖表也會隨之更新以反映所選的結束日期。
  • 搜尋 (Search): 使用文字框過濾表格中的代碼或公司名稱。
  • 自定義欄位 (Customize Columns): 點擊可切換顯示或通過拖放重新排序欄位。您也可以重命名欄位標題。
  • 管理代碼 (Manage Tickers): 新增或移除主要區塊中的代碼。您也可以重命名區塊標題。
  • 互動標題: 點擊欄位標題可進行排序。點擊 Date 標題可更改日期顯示格式。點擊 Since... 標題可更改該欄位的參考起始日期。
  • 點擊行: 點擊任意行將打開該特定代碼的詳細圖表。

show_chart 圖表 (Chart Tab)

此頁面允許對股票表現進行詳細的時間比較分析。

導航：

  • 子分頁: 使用頂部的按鈕（如 1-Week View, Main, Peers）切換不同的預設圖表視圖。雙擊分頁可重命名。

控制項：

  • 代碼輸入: 編輯以逗號分隔的代碼列表，以更改圖表上顯示的股票。
  • 時間控制: 選擇預設範圍（1週、1月、3月、年初至今等）或選擇「自定義 (Custom)」以定義特定開始和結束日期。
  • 更新按鈕: 更改代碼或日期後必須點擊此按鈕以刷新圖表。

圖表類型：

  • 指數化價格: 大多數分頁顯示重定基數的表現（從 0% 開始），便於比較。
  • 交易量 (Chart-7): 一個專門的網格視圖，顯示最多 4 個代碼的價格與成交量。

article 新聞 (News Tab)

訪問觀察名單公司的最新新聞和 AI 生成摘要。

如何使用：

  • 選擇公司: 使用下拉選單選擇特定公司。選擇「全部 (All)」將匯總所有來源的新聞。
  • 過濾來源: 按提供商（如 Reuters, Bloomberg）縮小新聞範圍。
  • 日期範圍: 點擊以選擇開始和結束日期來過濾新聞項目。
  • AI 摘要: 如果可用，列表中的第一項將是近期事件的 AI 生成摘要。
  • 閱讀: 點擊左側的新聞項目，在右側窗格閱讀完整內容。

table_view 數據 (Data Tab)

查看並導出儀表板背後的原始數據。

  • 過濾: 設定開始和結束日期以限制數據範圍。
  • 樞紐視圖: 數據以日期為列、代碼為欄進行樞紐分析，便於分析。
  • 導出: 使用「導出 Excel」按鈕下載包含價格和成交量工作表的完整數據集。

email 報告生成 (Reporting)

生成適合電子郵件發送的每週報告。

  • 點擊頂部標題中的 Generate Report。
  • 系統將捕捉當前圖表狀態並編譯 AI 摘要。
  • 可編輯內容: 您可以點擊生成的預覽內容進行文字修改、刪除區塊或添加註釋，然後再下載。
  • 下載草稿: 您可以將報告下載為 Outlook .eml 文件以便稍後發送。
  • 編輯欄位: 下載前您可以預先填寫收件人、副本和主旨行。
