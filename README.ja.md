# 📈 デイリーマーケットアップデート ダッシュボード

[![Live Demo](https://img.shields.io/badge/Demo-Live_View-success?style=for-the-badge&logo=github)](https://andychu221.github.io/market-summary-dashboard/)

株式アナリストや投資家向けに設計された、包括的なフロントエンド金融ダッシュボードです。リアルタイムのパフォーマンス追跡、インタラクティブなチャート、AIを活用したニュースのセンチメント分析、財務諸表、決算説明会の要約を、軽量な1つのWebアプリで提供します。

👉 **[ライブデモを見るにはここをクリック](https://andychu221.github.io/market-summary-dashboard/)**

## 🗄️ データソースとアーキテクチャ

このプロジェクトは**サーバーレス・フロントエンド・アーキテクチャ**を採用しています。従来のバックエンドサーバーに依存せず、Webアプリが直接公開APIを呼び出し、自動化されたGitHubリポジトリから毎日更新される静的JSONデータを動的に読み込みます。

* **市場・マクロデータ:** Yahoo FinanceおよびFREDから取得。日々の株価、移動平均線、時価総額、トータルリターン、マクロ経済イベントのカレンダーを含みます。
    * *ソース:* `andychu221/yhfinance-fred-data` リポジトリ
* **ニュースとセンチメント:** ロイター、WSJ、FT、聯合報、自由時報からの情報を集約。関連性スコア、センチメントスコア、事前処理されたAI要約が含まれています。
    * *ソース:* `andychu221/reuters-news-archive` リポジトリ
* **決算説明会（アーニングスコール）:** 対象企業の決算発表スケジュールと書き起こし（トランスクリプト）の全文。
    * *ソース:* `andychu221/Earnings-Call-Transcripts` リポジトリ
* **財務諸表:** 米国株の損益計算書、貸借対照表、キャッシュフロー計算書の四半期および年次データ。
    * *ソース:* `yhfinance-fred-data/us_financials` フォルダ
* **リアルタイムAI統合:** **Google Gemini API** (`gemini-2.5-flash-lite`) に直接接続。ユーザーが選択したニュースやチャート期間の文脈をAPIに送信し、リアルタイムのテクニカル分析やポートフォリオの要約を生成します。

## ✨ 主な機能

* 📊 **パフォーマンス追跡:** 業界セグメントごとに分類されたカスタムウォッチリストを監視。「リターンビュー」と「テクニカルビュー（移動平均、1年間のレンジ）」の切り替えに対応。
* 🤖 **AIによる分析:** Google Gemini APIをワンクリックで呼び出し、複雑なニュースや決算説明会から重要なインサイトを抽出します。
* 📈 **インタラクティブなチャート:** Chart.jsを使用し、休場日を排除した正確な取引日基準のX軸を持つ動的チャートを描画します。
* 📰 **スマートニュースダッシュボード:** 過去5日間のニュース量トレンド、センチメント統計、情報源の分布、キーワード統計を備えています。
* 💵 **インタラクティブな財務諸表:** 行のドラッグ＆ドロップカスタマイズや、成長率（Growth）と利益率（Margin）の自動計算をサポートしています。

## 🚀 使い方

1.  **アプリを開く:** [ライブデモ](https://andychu221.github.io/market-summary-dashboard/)にアクセスするか、ローカルで`index.html`を実行します。
2.  **AI機能の有効化:**
    * 右上の ⚙️ (設定) アイコンをクリックします。
    * **Google Gemini APIキー** を入力します（安全のためローカルブラウザのみに保存されます）。
    * 好みのAI出力言語（英語 / 繁体字中国語）を選択します。
