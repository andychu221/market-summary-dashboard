# 📈 デイリーマーケットアップデート ダッシュボード

株式アナリストや投資家向けに設計された、包括的なフロントエンド金融ダッシュボードです。リアルタイムのパフォーマンス追跡、インタラクティブなチャート、AIを活用したニュースのセンチメント分析、決算説明会の要約を、軽量な1つのWebアプリで提供します。

## ✨ 主な機能

*   📊 **パフォーマンス追跡:** 業界セグメント（ファウンドリ、ファブレス、製造装置、メモリなど）ごとに分類されたカスタムウォッチリストを監視します。リターンビューとテクニカルビュー（移動平均、1年間のレンジ）の切り替えに対応しています。
*   🤖 **AIによる分析:** Google Gemini APIと統合されており、市場トレンドの自動要約、テクニカルチャートの分析、複雑な決算説明会の書き起こしからの重要なインサイト抽出を行います。
*   📈 **インタラクティブなチャート:** Chart.jsを使用して、動的な2x2グリッド（1週間、1ヶ月、年初来、1年）やカスタム期間の比較チャートを描画します。X軸は実際の取引日に正確に合わせて配置されています。
*   📰 **スマートニュースダッシュボード:** 過去5日間のニュース量トレンド、全体的なセンチメント分析（ポジティブ/ニュートラル/ネガティブ）、情報源の分布、トップキーワード統計を備えています。
*   📅 **決算・マクロカレンダー:** 今後の決算発表やマクロ経済イベントを週間カレンダー形式で視覚化します。

## 🛠️ 技術スタック

*   **フロントエンド:** HTML5, CSS3, Vanilla JavaScript
*   **ライブラリ:** [Chart.js](https://www.chartjs.org/) (データ可視化), [Marked.js](https://marked.js.org/) (Markdown解析)
*   **AIモデル:** Google Gemini (gemini-2.5-flash-lite)
*   **データソース:** GitHubリポジトリ（ロイターニュースアーカイブ、Yahoo Finance/FREDデータ、決算書き起こし）からJSONデータを動的に取得します。

## 🚀 使い方

1.  **リポジトリのクローン:**
    ```bash
    git clone [https://github.com/YourUsername/YourRepoName.git](https://github.com/YourUsername/YourRepoName.git)
    ```
2.  **アプリの実行:**
    `index.html`をダブルクリックするだけで、最新のWebブラウザで開くことができます。ローカルサーバーやバックエンドのセットアップは不要です。
3.  **AI機能の有効化:**
    *   右上の ⚙️ (設定) アイコンをクリックします。
    *   **Google Gemini APIキー** を入力します。
    *   好みのAI出力言語（英語 / 繁体字中国語）を選択します。
